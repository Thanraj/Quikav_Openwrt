Quikscan

quikscan is QuikAV's command line virus scanner. It can be used to scan files and/or directories for viruses. In order for quikscan to work proper, the QuikAV virus database files must be installed on the system you are using quikscan on.

The general usage of quikscan is: quikscan [options] [file/directory/-]

For more detailed help, type 'man quikscan' or 'quikscan -help'.
