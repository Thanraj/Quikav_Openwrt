Quikconf

quikconf is the Quik Anti-Virus configuration utility. It is used for displaying values of configurations options in QuikAV, which will show the contents of quikd.conf (or tell you if it is not properly configured), the contents of freshquik.conf, and display information about software settings, database, platform, and build information.
